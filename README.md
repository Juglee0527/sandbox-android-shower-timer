# 🚿 Music Shower Timer
> 음성 안내 기반 샤워 타이머 & YouTube 연동 안드로이드 앱  

샤워 중 시간을 확인하기 위해 화면을 볼 필요 없이,  
**음성으로 경과 시간을 안내**해주고 **YouTube와 연동하여 음악을 재생**할 수 있는 스마트 타이머 앱입니다.

---

## 📌 프로젝트 개요

- **과목**: 스마트모바일프로그래밍
- **개발환경**: Android Studio
- **개발언어**: Java
- **플랫폼**: Android
- **프로젝트 성격**: 개인 프로젝트
- **목표**: 샤워 시간을 효율적으로 관리하고 몰입감 있는 샤워 환경 제공

---

## 💡 개발 배경

- 샤워 시간이 길어지는 습관 개선 필요
- 샤워 중 화면 조작의 불편함(젖은 손)
- 샤워 중 음악 재생 수요에 대한 통합 기능 부재
- 기존 스톱워치는 “시간 표시”만 있고 “음성 안내” 기능이 부족함

> 위 문제를 해결하기 위해  
> `타이머 + 음성 안내 + YouTube 연동` 기능을 하나의 앱으로 구현하고자 함

---

## ✅ 주요 기능

| 기능 | 설명 |
|------|------|
| 타이머 기능 | 샤워 시간 측정을 위한 스톱워치 / 타이머 |
| 음성 안내(TTS) | 사용자가 설정한 시간 간격마다 “n분 지났습니다” 음성 안내 |
| YouTube 연동 | 앱 내 버튼 클릭 시 YouTube 실행 (노래/플레이리스트 재생) |
| UI 표시 | 경과 시간 / 남은 시간 시각적으로 표시 |
| 기록 기능(Optional) | 이전 샤워 시간 기록 저장 |

---

## 🛠 기술 스택

- Java
- Android Studio
- TextToSpeech (TTS)
- Intent (YouTube 연동)
- CountDownTimer / Handler
- SharedPreferences (선택)

---

## 📱 화면 구성 (예정)

1. 메인 화면
   - 타이머 표시
   - Start / Stop / Reset 버튼
   - "Open YouTube" 버튼

2. 설정 화면
   - 목표 시간 설정
   - 음성 알림 간격 설정
